NORZEC · Relatório Integrado Corredor Beira (Abril-Maio/2026)

Este projeto consiste em uma ferramenta de análise climática focada no Corredor da Beira, em Moçambique, abrangendo as regiões de Beira, Chimoio e Tete. 
O sistema processa dados térmicos regionais para gerar insights retrospectivos e modelos prescritivos voltados à tomada de decisão operacional.  
🚀 Visualização OnlineO portfólio interativo gerado por este sistema está disponível via GitHub Pages:👉 https://norisfrancisco.github.io/Relatorio_Mensal_Corredor_Beira/

📊 Visão Geral do ProjetoO relatório analisa o comportamento térmico do corredor durante o mês de abril de 2026 e projeta as tendências e riscos operacionais para maio de 2026. 
Funcionalidades Principais:Processamento de Séries Temporais: Análise de 30 dias de dados térmicos com 100% de consistência e sem lacunas (gaps).  
Visualização Estatística: Geração de gráficos de tendência sazonal, boxplots de comparação regional e histogramas de previsão.  
Análise de Gradiente: Monitoramento da variação térmica costa-interior, identificando Tete como o ponto crítico (34.3°C de média máxima em abril).  
Portfólio Automático: Exportação de um relatório em HTML com imagens otimizadas em Base64 para carregamento instantâneo.

⚠️ Recomendações Prescritivas (Maio 2026)
Com base na análise de tendência, o sistema identificou pontos críticos para o planejamento regional:
Logística: Janela ótima para transporte de cargas sensíveis identificada entre 01 e 10 de maio.  
Agricultura: Período ideal para plantio devido à subida da umidade relativa entre 01 e 05 de maio.  
Alerta de Saúde: Risco de doenças respiratórias devido à queda abrupta de temperatura em Chimoio (22°C para 14°C) prevista para o período de 11 a 15 de maio.
